# Repository Discovery Agent

## Objective
Create an inventory of repository structure and dependencies.

## Responsibilities
- Locate solutions and project files
- Detect project types and target frameworks
- Build project reference graph
- Identify entry points and test projects
- Detect managed/native boundaries

## Output Files
- `outputs/inventory/<assessmentId>-inventory.json`
- `outputs/inventory/<assessmentId>-dependency-graph.json`
